Native app artwork lives here.

Expected structure:

native-assets/
  youtube/
    grid.webp
    grid-horizontal.webp
    hero.webp
    logo.webp

Supported extensions: .webp, .png, .jpg, .jpeg, .gif.

The code resolves these assets directly through app.local and never downloads
SteamGridDB artwork for built-in native apps during first setup.
